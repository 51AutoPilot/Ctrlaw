# Mission Control - AI Operating System

## Project Overview
A Next.js + Convex dashboard for managing AI agent operations.

## Systems to Build

### 1. Tasks Board
- Track all tasks with status (todo, in-progress, done, blocked)
- Assign to human or AI
- Real-time updates
- URL: /tasks

### 2. Calendar
- Show all scheduled tasks/cron jobs
- Audit trail of executions
- URL: /calendar

### 3. Memory
- Searchable memory documents
- Full-text search
- URL: /memory

### 4. Content Pipeline
- Idea → Script → Thumbnail → Filming → Publish
- Drag & drop stages
- URL: /content

### 5. Team
- Sub-agents organization
- Roles & responsibilities
- Current tasks
- URL: /team

### 6. Office
- Real-time agent status
- Avatar + work area visualization
- URL: /office

## Tech Stack
- Next.js 14 (App Router)
- Convex (Database + Realtime)
- TailwindCSS
- TypeScript
